# Repository Scope

## Purpose

This repository exists to present StageScout publicly without open-sourcing the implementation. It is meant to show the product, the system design, and the deployment approach behind the live app.

## Included

- High-level product overview
- Architecture and infrastructure documentation
- Public-safe diagrams and screenshots
- Stack, flow, and deployment summaries

## Excluded

- Application source code
- Private infrastructure code
- Internal scripts and automation
- Secrets, credentials, and environment files
- Internal debugging endpoints, operational notes, and production-sensitive details

## Publishing rule

Only add material that is safe to treat as permanent public documentation. If a file would help someone rebuild the exact system, access infrastructure, or infer secrets and internal operations, it does not belong in this repository.
